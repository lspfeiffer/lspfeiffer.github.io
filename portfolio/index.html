QuantSoftware Toolkit

QuantSoftwareToolKit (QSTK) is an open source python library designed to support portfolio construction and management. Features include portfolio optimization, technical indicator analysis, portfolio analysis tools, event studies, and visualization tools. QSTK has had more than 100,000 downloads.

Recommendation Systems

Created a music rating prediction system, which predicted unknown ratings using collaborative filtering SVD. Worked on the Yahoo music dataset, which has 1 million users and more than 600,000 unique music items. Also created a movie recommendation system using user and item similarities using the movielens dataset.

GT EventConnect

Created web and android apps for events on Georgia Tech campus. Features include personalized recommendation of events to the user based on department, college, location and user history. The apps also provided a news feed of upcoming and latest events on campus.

Github Visualizer

D3.js based visualization tool for analyzing data and activity of users in an organization. Features include analyzing contribution and activity trends between different users or different projects over various periods of time. Can be used for decisions such as manpower allocation and performance reviews.

PageRank

Implemented multiple versions of map-reduce algorithms to calculate PageRank for the SNAP MemeTracker and english wikipedia datasets using Hadoop. Analyzed and compared the results and performance with the standard implementation of Apache Giraph to better understand the tradeoffs of using Giraph.

Dimensionality Reduction

A comparative study between PCA, KPCA and ICA compression algorithms for dimensionality reduction for classification tasks. Applications studied were Face Recognition, Handwriting Recognition and Arrhythmia detection. KPCA improves the accuracy of the classifier but has an increase in training time.
